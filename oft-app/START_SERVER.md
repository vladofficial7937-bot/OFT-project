# Запуск dev сервера

## Команда для запуска:

```bash
cd "D:\OFT demo versions\oft-app"
npm run dev
```

## После запуска:

1. Откройте браузер
2. Перейдите по адресу, который покажет Vite (обычно `http://localhost:5173`)
3. Если порт занят, Vite автоматически выберет следующий свободный порт

## Возможные проблемы:

### Порт занят
Если порт 5173 занят, Vite использует следующий свободный (5174, 5175 и т.д.)
Смотрите в консоли, какой порт был выбран.

### Ошибки компиляции
Если есть ошибки TypeScript, они будут показаны в консоли.
Исправьте ошибки и перезапустите сервер.

### Кэш браузера
Если страница не обновляется, попробуйте:
- Hard refresh: Ctrl+Shift+R (Windows) или Cmd+Shift+R (Mac)
- Очистить кэш браузера

## Проверка работы:

1. Откройте DevTools (F12)
2. Проверьте вкладку Console на наличие ошибок
3. Проверьте вкладку Network - должны загружаться файлы без ошибок
