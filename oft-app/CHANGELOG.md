# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.1.0] - 2025-01-20 - MVP Release

### Added
- ✅ **Режим "Тренер"**: Dashboard с списком клиентов, добавление клиентов через wizard, profiles клиентов


- ✅ **Режим "Клиент"**: Главная страница с приветствием, карта мышц, каталог упражнений
- ✅ **Выполнение тренировок**: Чек-лист упражнений, запись подходов и повторений, таймер отдыха
- ✅ **Страница прогресса**: Статистика тренировок, календарь активности, история тренировок
- ✅ **ИИ-заглушки**: Архитектура для будущей интеграции ИИ-агентов (диагностика, генерация планов, советы)
- ✅ **Адаптивный дизайн**: Полная поддержка mobile (375px), tablet (768px), desktop (1024px+)
- ✅ **Тёмная тема**: Кастомная цветовая схема с красным акцентом (#ff4444)
- ✅ **Toast уведомления**: Система уведомлений для успешных операций и ошибок
- ✅ **Плавные анимации**: Fade-in, scale-in, slide-up анимации для улучшения UX
- ✅ **LocalStorage**: Полноценное хранилище данных с CRUD операциями
- ✅ **20+ упражнений**: База упражнений с описанием техники, советами и рекомендациями
- ✅ **7 групп мышц**: Полная классификация мышечных групп
- ✅ **Демо-данные**: Seed data для быстрого старта
- ✅ **Error Boundary**: Обработка ошибок React компонентов
- ✅ **Code Splitting**: Lazy loading страниц для оптимизации bundle
- ✅ **PWA готовность**: Manifest.json для установки как приложение
- ✅ **SEO оптимизация**: Meta tags, Open Graph, robots.txt

### Technical
- React 18 + TypeScript
- React Router v6 для навигации
- Zustand для state management
- Tailwind CSS для стилизации
- Vite для сборки
- LocalStorage для хранения данных

### Known Issues
- ❌ Нет авторизации (MVP ограничение)
- ❌ Данные только локальные (нет backend)
- ❌ ИИ работает с заглушками (требуется подключение API)
- ❌ Видео упражнений — плейсхолдеры
- ❌ Нет экспорта/импорта данных
- ❌ Нет мультиязычности
