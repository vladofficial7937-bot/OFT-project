# Устранение проблем

## Сайт не открывается

### 1. Проверка сервера

Убедитесь, что dev сервер запущен:
```bash
cd "D:\OFT demo versions\oft-app"
npm run dev
```

Должно появиться сообщение:
```
  VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 2. Проверка браузера

1. Откройте браузер (Chrome, Firefox, Edge)
2. Перейдите на `http://localhost:5173`
3. Если не открывается, попробуйте другой порт (5174, 5175 и т.д.)

### 3. Проверка консоли

Откройте DevTools (F12) и проверьте:

**Console (Консоль):**
- Есть ли красные ошибки?
- Если есть, скопируйте текст ошибки

**Network (Сеть):**
- Загружаются ли файлы (main.tsx, App.tsx и т.д.)?
- Есть ли ошибки 404 или 500?

### 4. Типичные проблемы

#### Проблема: Белый экран
**Решение:**
- Проверьте консоль на ошибки JavaScript
- Проверьте, что все зависимости установлены: `npm install`

#### Проблема: "Cannot find module"
**Решение:**
- Удалите `node_modules` и `package-lock.json`
- Запустите `npm install` заново

#### Проблема: Порт занят
**Решение:**
- Остановите другие процессы на порту 5173
- Или используйте другой порт: `npm run dev -- --port 3000`

#### Проблема: Ошибки компиляции TypeScript
**Решение:**
- Проверьте консоль терминала на ошибки
- Исправьте ошибки TypeScript

### 5. Очистка кэша

```bash
# Очистить node_modules
rm -rf node_modules
rm package-lock.json

# Переустановить зависимости
npm install

# Очистить кэш Vite
rm -rf node_modules/.vite

# Запустить заново
npm run dev
```

### 6. Проверка файлов

Убедитесь, что все файлы на месте:
- `src/main.tsx` - точка входа
- `src/App.tsx` - главный компонент
- `src/index.css` - стили
- `index.html` - HTML шаблон

### 7. Логи для отладки

Если проблема сохраняется, пришлите:
1. Текст ошибки из консоли браузера (F12 → Console)
2. Текст ошибки из терминала (где запущен `npm run dev`)
3. Скриншот страницы (если возможно)
